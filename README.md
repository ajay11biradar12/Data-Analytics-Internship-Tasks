# Data Analytics Internship Tasks

This repository contains two major tasks completed during my Data Analytics internship. These projects demonstrate my skills in Machine Learning, Time-Series Analysis, and Data Visualization.

## 🚀 Projects Overview

### 1. Customer Churn Prediction
- **Goal:** Predict which customers are likely to stop using a service.
- **Models Used:** Logistic Regression, Random Forest, and XGBoost.
- **Key Techniques:** Data Cleaning, Feature Encoding, Scaling, and Model Evaluation (Accuracy, Recall, ROC-AUC).
- **Outcome:** Successfully identified key drivers of churn such as 'Tenure' and 'Monthly Charges'.

### 2. Sales Forecasting
- **Goal:** Predict future sales trends based on historical retail data.
- **Models Used:** Facebook Prophet.
- **Key Techniques:** Time-Series Splitting, Seasonality Analysis (Weekly & Yearly), and Trend Identification.
- **Outcome:** Built a robust forecast model that accounts for promotional holidays and seasonal peaks.

## 🛠️ Tech Stack
- **Languages:** Python
- **Libraries:** Pandas, NumPy, Scikit-learn, XGBoost, Prophet, Matplotlib, Seaborn
- **Environment:** Jupyter Notebook

## 📈 Results
The projects include visual evaluations like Confusion Matrices, Feature Importance plots, and Forecast vs. Actual sales charts.
